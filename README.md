# Sistema de Recomendación de Películas con KNN

Proyecto desarrollado en Go que utiliza el algoritmo K-Nearest Neighbors (KNN) para recomendar películas en función de las preferencias del usuario.

## Características
- Implementado en Go.
- Uso de archivos CSV para almacenar datos.
- Clasificación mediante algoritmo KNN.
- Recomendación de películas basada en similitud.

## Archivos del proyecto
- main.go: Código principal.
- peliculas.csv: Base de datos de películas.
- modelo.knn: Modelo entrenado.
- go.mod: Dependencias del proyecto.

## Ejecución

```bash
go run main.go
